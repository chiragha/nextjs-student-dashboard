## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/student-dashboard.git
cd student-dashboard
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Run Development Server

```bash
npm run dev
```

🌍 Open: `http://localhost:3000`

---

## 🏗️ Production Build

```bash
npm run build
npm run dev
```
